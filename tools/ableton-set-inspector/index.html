ABOUT THIS BUILD

        
Designed as a safe first-pass project report.

      

      

        

          
What it does

          
Reads the gzip-compressed XML inside an Ableton Live Set and turns recognisable structures into a human-readable report.

        

        

          
What it does not do

          
It cannot hear the project, verify licences, scan installed plug-ins, or prove that external file paths still exist.

        

        

          
Compatibility

          
The parser is deliberately tolerant of unknown nodes. New or unusual Live structures may produce a partial report rather than failing completely.
